A lightweight DOM snapshot and replay toolkit. `snapshot.js` serialises a live DOM tree into a portable JSON representation — capturing element structure, attributes, inline styles via CSSOM, input values, canvas pixels, and scroll positions. `rebuild.js` takes that JSON and reconstructs the DOM into a target document. `visualiser.html` provides a browser-based viewer: paste a snapshot JSON on the left, hit Render, and the right-hand iframe replays the captured page as a sandboxed static reconstruction with no scripts running.
